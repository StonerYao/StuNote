Vue 笔记

Vue 安装

  • 官网下载或 CDN，通过 <script> 引入。
    Vue 会被注册为一个全局变量。
  • 通过 NPM 安装。
    npm install vue
  • 通过 Vue CLI 安装。

Vue 版本

  • 开发版本 vue.js
    包含完整的警告和调试模式。
  • 生产版本 vue.min.js
    没有警告并使用了压缩。

Vue 模板语法

Vue.js 使用了基于 HTML 的模板语法。在底层的实现上，Vue 将模板编译成虚拟 DOM 渲染函数。也可直接写渲染 (render) 函数，使用可选的 JSX 语法。

插值

使用“Mustache”语法 {{ }} 实现数据绑定。其中的值可以是以下列表中的任意一项：

  • 文本 {{ msg }}
    Mustache 标签将会被替代为对应数据对象上 msg property 的值，如果没有对应的 msg property 就会报错。无论何时，绑定的数据对象上 msg property 发生了改变，插值处的内容都会更新。
  • JavaScript 数据值 {{ 1 }} | {{ 'ok' }} | {{ true }} | {{ undefined }} | {{ null }} | {{ ['a', 'b', 'c'] }} | {{ {height: '140', weight: '100'} }}
  • JavaScript 表达式 {{ number + 1 }} | {{ ok ? 'YES' : 'NO' }} | {{ message.split('').reverse().join('') }}
    JavaScript 表达式会在所属 Vue 实例的数据作用域下作为 JavaScript 被解析。有个限制就是，每个绑定都只能包含单个表达式。

Vue 实例

引入 vue.js 后，它提供了一个构造函数 Vue，通过用 new Vue() 创建一个 Vue 实例，同时可以传入一个选项对象来控制行为，最后再用变量接住这个实例，以此搭建 Vue 应用。

选项/DOM

el

类型：string | Element
限制：只在用 new 创建实例时生效。
作用：配置 Vue 实例要控制的区域（元素）。
关联：vm.$el vm.$mount()
关键字：#挂载目标#

选项/数据

data

类型：Object | Function
限制：组件的定义只接受 function。
作用：配置 Vue 实例要用到的数据对象。
关联：vm.$data.([property]) vm.[property]
关键字：#数据对象# #响应式#

实例方法/生命周期

vm.$mount( [elementOrSelector] )

参数：
{Element | string} [elementOrSelector]
{boolean} [hydrating]
返回值：vm - 实例自身
作用：配置 Vue 实例要控制的区域（元素）。
关联：el
关键字：#手动挂载# #延迟挂载#

Vue 示例

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Vue</title>
  <!-- 引入vue.js 提供一个构造函数Vue -->
  <script src="./vue.js" type="text/javascript" charset="utf-8"></script>
</head>
<body>
  <!-- 被Vue控制的区域 模板 -->
  <div id="app">
    <!-- 插值表达式 获取Vue中的数据对象 -->
    {{ a }}
  </div>
  <!-- Vue脚本 -->
  <script type="text/javascript">
    const vm = new Vue({ // 创建Vue实例 传递配置对象
      el: '#app', // 控制id为app的元素
      data: { // 配置要用到的数据对象
        a: 'Hello Vue!',
        b: {
          ba: 111,
          bb: '222'
        }
      }
    });
  </script>
</body>
</html>
